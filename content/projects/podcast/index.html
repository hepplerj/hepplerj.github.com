Podcast Tools

Show Notes

To create show notes more easily, I've created a small JavaScript bookmarklet. To use:

  1. Drag this link to your bookmarks bar.
  2. Visit a page you want to include in show notes. Highlight an excerpt of text.
  3. Click the bookmarklet. A popup window will appear with the page title, URL, and highlighted text formatted as Markdown.
  4. Copy the text from the window and click "OK" or "Cancel." Paste the text into the show notes document.
